About this project:

This Pokédex web was designed and programmed by Jose in Australia (see Instagram or Github).

The main purpose of this project is to practice Javascript, especially Async JS, use of Fetch API and consume an external API. This project was developed using HTML, Tailwind.css, and Vanilla JS.
